# Alchimisti Anonimi

> **Academic project** — Software Engineering course  
> University of L'Aquila · A.Y. 2025/2026

Exam project for the **Software Engineering** course.
A Java implementation of the domain logic for the board game **"Alchemists"** by Matúš Kotry (CGE).

---

## Purpose & Learning Objectives

This project was designed to develop and assess the following software design competencies:

- Apply **OOA** **OOD** and **GRASP patterns** to a real domain model
- Translate **conceptual models** and **System Sequence Diagrams (SSDs)** into working Java code (OOA/OOD -> OOP)
- Design a clean **layered architecture** separating controller, model, and service responsibilities 
- Practice **domain-driven decomposition** of a non-trivial rule system
